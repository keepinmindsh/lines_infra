# Terraform 

테라폼은 코드로 인프라를 관리할 수 있는 도구 입니다. 

## Hello! Terraform

### MacOS 기준 Terraform 

- Terraform 설치하기 

```shell
$ brew install terraform 
```

- Terraform 버전체크 

```shell
$ terraform -version
Terraform v1.3.8
on darwin_arm64
```

- GCP Terraform 계정 만들기 

GCP의 Service Account을 Terraform을 기준으로 생성한다.
